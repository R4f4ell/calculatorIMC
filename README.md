# Calculadora de IMC

Projeto desenvolvido com foco em responsividade, acessibilidade, boas práticas de código e performance.  
Permite que o usuário insira sua altura e peso para calcular o IMC, com resultado exibido em tempo real e classificação automática.

## 🔧 Tecnologias

- React
- Vite
- SCSS
- JavaScript 

## 🚀 Funcionalidades

- Cálculo automático do IMC com base na altura e peso
- Limpeza rápida dos campos com botão "Limpar"
- Exibição da classificação do IMC (ex: Magreza, Normal, Sobrepeso...)
- Design responsivo e adaptado a diferentes dispositivos
- Acessibilidade com uso de `aria-label`, `aria-labelledby`, `label` semântico
